# Install & run TreeMark (PyCharm / local)

1. Clone repo
   - git clone <your-repo-url>
2. Open in PyCharm
   - File -> Open -> select project folder
   - Create a virtual environment (Poetry or venv)
3. Using Poetry:
   - poetry install
   - poetry run treemark generate /path/to/folder
